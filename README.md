# degen_art

An abstract art application, written in raw Win32 C++.

Modified from a Charles Petzold source.

### Why is it named Degen Art
It is named degen art, as a way to make fun of Hitler.
Huh? Why? He famously called many new artists and art forms, including abstract/surrealist art, "degenerate art".

Many artists died as a result, and there is artwork that has been lost because of it.
I couldn't think of a name, so, well there you go. My dark humour prevails lol.
